# Title: NEAT Algorithm to Play Simple Game

## Project Description
For my project I want to use neuroevolution of 
augmenting topologies (NEAT) to train a neural 
network to learn how to play a simple game. NEAT is an 
evolutionary algorithm that creates a neural network.

## Project Goals
1. Create simple game 
2. Train neural network that can successfully play game
3. Track evolution of neural network overtime in both the physical neural network and how it plays the game.
