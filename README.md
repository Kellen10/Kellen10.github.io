# Title: NEAT Algorithm to Play Simple Game

## Project Description
I will use NeuroEvolution of Augmenting Topologies (NEAT) to train a neural network to play a simple game. NEAT works by evolving populations of neural networks over multiple generations, similar to how biological evolution works. The basic idea is a population of neural networks is created, and each network plays the game and receives a fitness score based on their performance. The best-performing networks are selected to "reproduce," and through mutation and crossover a new generation of neural networks will be created. This process repeats, with each generation typically performing better than the last. For this project, I'll either create a simple game from scratch or adapt an existing one that's straightforward enough for the NEAT algorithm. This project will be fun and educational because it will create an AI to play a game from scratch and the evolving neural network can be observed. The goal is to have the trained neural network play the game at least as well or better than a human player.



## Project Goals
1. Create simple game 
2. Train neural network that can successfully play game
3. Track evolution of neural network overtime in both the physical neural network and how it plays the game.
